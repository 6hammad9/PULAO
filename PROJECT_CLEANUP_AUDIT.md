## Current Confusion Source

The app has separate concepts that currently live beside each other:

- source code
- generated gallery/evidence files
- event-scoped OCR data
- old training/test image datasets
- backup Python experiments
- local logs and caches

The best cleanup path is to keep active source in place, ignore generated runtime data, and move legacy/backup experiments into a single `archive/` folder or delete them after confirmation.
